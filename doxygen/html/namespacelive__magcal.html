Detailed Description

Live magnetometer calibration-coverage viewer.

Run nav_filters (writing sparxe_sensors_raw.csv) in one terminal, this in
another, then pick the robot up and tumble it through all orientations. The
three projections fill in live; the title turns green when the point cloud
covers the sphere well enough to calibrate. Then run calibrate.py on the log.

    python live_magcal.py [path_to_sparxe_sensors_raw.csv]

Press 'q' to quit.
