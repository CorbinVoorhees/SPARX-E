UWB Pi gateway Ubuntu installer

This package installs the Raspberry Pi 5 / Ubuntu side of the Qorvo DWM3001CDK setup. The Pi runs the UWB board as the FiRa initiator/controller and exposes a simple serial API to the Arduino. The Arduino prints the result to the Arduino Serial Monitor.

Files you need on the Pi

Copy these to the Pi:

  1. This folder or zip contents.
  2. The Qorvo SDK zip, named SDK(1).zip or SDK.zip, placed beside install_pi_gateway.sh.

The SDK itself is not bundled in this package because it is large.

Hardware serial requirement for Arduino Serial Monitor output

To see range data in the Arduino Serial Monitor, the Arduino needs two serial paths:

  • Serial over Arduino USB -> Arduino Serial Monitor.
  • Serial1 -> Raspberry Pi serial link.

Use an Arduino with Serial1, such as a Mega, Leonardo-class board, Nano Every, SAMD board, ESP32, or equivalent. If using Pi GPIO UART, level-shift signals if the Arduino is 5 V.

One-command install

unzip uwb_pi_ubuntu_install.zip
cd uwb_pi_ubuntu
sudo ./install_pi_gateway.sh --start

If auto-detection cannot find the SDK zip or serial ports:

sudo ./install_pi_gateway.sh \
--sdk-zip /path/to/SDK\‍(1\‍).zip \
--uwb-port /dev/serial/by-id/<PI_UWB_PORT> \
--arduino-port /dev/serial/by-id/<ARDUINO_LINK_PORT> \
--start

What the installer does

  • Installs Python, venv, pip, unzip, screen, and minicom.
  • Unpacks the Qorvo SDK to /opt/qorvo_uwb_sdk.
  • Creates a Python virtual environment under SDK/Tools/uwb-qorvo-tools/.venv.
  • Installs pyserial and colorama.
  • Installs uwb_gatewayd.py to /opt/qorvo_uwb_pi_gateway/bin.
  • Installs the Arduino sketch to /opt/qorvo_uwb_pi_gateway/arduino/UwbRangeMonitor.
  • Creates /etc/default/qorvo_uwb_pi_gateway.
  • Creates and enables qorvo-uwb-pi-gateway.service.

Service commands

sudo systemctl start qorvo-uwb-pi-gateway.service
sudo systemctl stop qorvo-uwb-pi-gateway.service
sudo systemctl restart qorvo-uwb-pi-gateway.service
journalctl -u qorvo-uwb-pi-gateway.service -f

Arduino usage

Upload the included sketch:

/opt/qorvo_uwb_pi_gateway/tools/upload_arduino_sketch.sh --help

Open Arduino Serial Monitor at 115200 baud and type:

r

The Arduino sends GET_RANGE to the Pi over Serial1; the Pi returns the latest range; the Arduino prints it to the monitor.

Expected monitor output:

UWB: RANGE ok=1 peer=01:00 seq=184 range_cm=243 range_m=2.430 rate_mps=-0.180 rate_valid=1 age_ms=31 status=Ok

Default UWB role/config

  • Role: initiator / controller
  • Local MAC: 0x0
  • Peer/responder MAC: 0x1
  • Session ID: 42
  • Channel: 9
  • DS-TWR deferred, SP3, 64-symbol STS, 2 ms slots, 200 ms ranging interval, 25 slots per ranging round.

These values must match the ground-station responder.
