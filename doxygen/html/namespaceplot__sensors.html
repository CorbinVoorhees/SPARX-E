plot_sensors.load_accel_q0 (   base) 
Return (t_abs, a_q0 Nx3, bias_q0 Nx3) from the TRANS-EKF log.

a_q0 = raw accel rotated into the q0 frame (done in nav_node), bias_q0 is
the accel bias state (already q0). unbiased = a_q0 - bias_q0 should sit at
the inertial gravity (grav_ref) at rest and stay put during pure rotation.

◆ load_trans_accel()

plot_sensors.load_trans_accel (   base) 
Return (t_abs, ax_raw Nx3, ba_body Nx3) from the TRANS-EKF log.

ax_raw (body-frame raw accel) and ba_body (accel bias already rotated into
the body frame in nav_node) live on the same rows, so unbiased = raw-ba_body
needs no interpolation. Returns None if the log/columns aren't present.
