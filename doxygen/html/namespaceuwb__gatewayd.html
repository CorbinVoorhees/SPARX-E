Detailed Description

Pi-side UWB gateway for Qorvo DWM3001CDK UCI firmware.

- Controls a DWM3001CDK over Qorvo UCI on USB/UART.
- Starts a FiRa DS-TWR deferred ranging session.
- Stores the latest range sample from SESSION_INFO_NTF.
- Answers Arduino serial commands such as GET_RANGE.

This file intentionally imports the Qorvo SDK Python UCI package from the SDK
source tree rather than copying Qorvo code into this project.
