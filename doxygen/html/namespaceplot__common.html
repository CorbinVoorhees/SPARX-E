Detailed Description

Shared interactivity helpers for the analysis plot scripts.

Provides:
  - enable_quit_key(*figs): press 'q' in any window to close everything.
  - add_time_window(fig, time_axes, t): a RangeSlider to view/zoom a section
    of time by setting the shared x-limits of the given time-series axes.

Add a slider that restricts the visible time span of `time_axes`.

fig          : the figure to attach the slider to.
time_axes    : iterable of Axes whose x-axis is time (seconds). All are
               driven together; axes that share an x-axis update as one.
t            : the time array (used only for its min/max bounds).
Returns the slider so the caller can keep a reference alive.
