 This package installs the Raspberry Pi 5 / Ubuntu side of the Qorvo DWM3001\+CDK setup. The Pi runs the UWB board as the Fi\+Ra initiator/controller and exposes a simple serial API to the Arduino. The Arduino prints the result to the Arduino Serial Monitor.\hypertarget{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md61}{}\doxysection{\texorpdfstring{Files you need on the Pi}{Files you need on the Pi}}\label{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md61}
Copy these to the Pi\+:


\begin{DoxyEnumerate}
\item This folder or zip contents.
\item The Qorvo SDK zip, named {\ttfamily SDK(1).zip} or {\ttfamily SDK.\+zip}, placed beside {\ttfamily install\+\_\+pi\+\_\+gateway.\+sh}.
\end{DoxyEnumerate}

The SDK itself is not bundled in this package because it is large.\hypertarget{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md62}{}\doxysection{\texorpdfstring{Hardware serial requirement for Arduino Serial Monitor output}{Hardware serial requirement for Arduino Serial Monitor output}}\label{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md62}
To see range data in the Arduino Serial Monitor, the Arduino needs two serial paths\+:


\begin{DoxyItemize}
\item {\ttfamily Serial} over Arduino USB -\/\texorpdfstring{$>$}{>} Arduino Serial Monitor.
\item {\ttfamily Serial1} -\/\texorpdfstring{$>$}{>} Raspberry Pi serial link.
\end{DoxyItemize}

Use an Arduino with {\ttfamily Serial1}, such as a Mega, Leonardo-\/class board, Nano Every, SAMD board, ESP32, or equivalent. If using Pi GPIO UART, level-\/shift signals if the Arduino is 5 V.\hypertarget{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md63}{}\doxysection{\texorpdfstring{One-\/command install}{One-command install}}\label{md_UWB_2uwb__pi__ubuntu_2README__PI__GATEWAY_autotoc_md63}
