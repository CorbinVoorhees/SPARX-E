\doxysubsection{Detailed Description}
\begin{DoxyVerb}Shared interactivity helpers for the analysis plot scripts.

Provides:
  - enable_quit_key(*figs): press 'q' in any window to close everything.
  - add_time_window(fig, time_axes, t): a RangeSlider to view/zoom a section
    of time by setting the shared x-limits of the given time-series axes.
\end{DoxyVerb}
